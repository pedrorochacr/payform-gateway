== Adicione o MultiPay ao WooCommerce ==

Este plug-in adiciona a plataforma de pagamento MultiPay ao WooCommerce.

Observe que o WooCommerce deve estar instalado e ativo.

MultiPay é uma Plataforma de pagamento brasileira desenvolvida pela empresa Multifidelidade Conta de Pagamentos LTDA.

O plugin MultiPay WooCommerce fornece um Checkout de Pagamento ágil e moderno, aceitando as principais formas de pagamento do mercado: PIX, cartão de crédito em até 6x sem juros, cartão de débito, ApplePay, Mercado Pago e PicPay.

Este é um produto Oficial Multifidelidade.

= Como Testar =

A API do MultiPay não possui um ambiente de testes. Todos os testes de pagamento devem ser feitos usando o ambiente de produção.

= Compatibilidade =

Compatível com as versões mais recentes do WooCommerce.

O uso do plugin [Brazilian Market on WooCommerce](http://wordpress.org/plugins/woocommerce-extra-checkout-fields-for-brazil/) é obrigatório, pois desta forma é possível enviar os campos de "CPF" ou "CNPJ" para processar o pagamento.


== Installation ==

= Requisitos Técnicos Mínimos =
* Versão mais recente do WooCommerce instalada e ativa.
* Faça o credenciamento em [Multifidelidade](https://multicrm.multifidelidade.app/Multi/CriarConta) para receber o seu "MerchantKey" e "EstablishmentCode".
* Ter instalado e configurado o plugin [Brazilian Market on WooCommerce](http://wordpress.org/plugins/woocommerce-extra-checkout-fields-for-brazil/).

= Instalação automática pelo administrador do WordPress: =

1. Faça login no painel de administração do WordPress.
2. Acesse "Plugins" no menu lateral de navegação do administrador do WordPress.
3. Uma vez dentro do plugin, clique em "Adicionar novo" e pesquise **'MultiPay - Pagamentos e Fidelidade para WooCommerce'** na lista de plugins do WordPress
4. Clique em "Instalar".
5. na seção "Plugins Instalados", ative o plug-in.
7. Em seguida, vá para as configurações do MultiPay e insira o **EstablishmentCode** e a **MerchantKey** que você recebeu no e-mail de confirmação.
7. Agora você verá no painel de configuração de pagamento do WooCommerce uma nova opção de pagamento chamada **"MultiPay"** que pode ser ativado.
8. Acesse a página Configurações do **MultiPay** para configurar seu formulário e ativar o **"MerchantKey"** e **"EstablishmentCode"** que recebeu apos o credenciamento.

== Frequently Asked Questions ==

= Qual é a licença do plug-in? =

* Este plugin é lançado sob uma licença GPL.

= 1. O que é o MultiPay? =

* O MultiPay é um gateway de pagamento para o WooCommerce que permite que os comerciantes aceitem várias formas de pagamento, incluindo PIX, cartão de crédito em até 6 vezes em juros, cartão de débito, Mercado Pago e PicPay. Ao realizar uma compra, o cliente acumula pontos que podem ser trocados por prêmios. Ele oferece uma integração fácil e segura para processar pagamentos em sua loja online.

= 2. Como faço para usar o MultiPay? =

* Para usar o MultiPay, você precisa se cadastrar na nossa empresa, a [Multifidelidade](https://news.multifidelidade.app/news/web/LandingPages/PluginApi/). Após o cadastro, você receberá dois códigos importantes: **EstablishmentCode** e **MerchantKey**. Esses códigos serão necessários para configurar o MultiPay no seu site WordPress.

= 3. Onde encontro o EstablishmentCode e a MerchantKey? =

* Após se cadastrar na **Multifidelidade**, você receberá um e-mail de confirmação contendo o **EstablishmentCode** e a **MerchantKey**. Certifique-se de guardar esses códigos em um local seguro, pois você precisará deles para configurar o MultiPay no seu site.

= 4. Quais são as formas de pagamento suportadas pelo MultiPay? =

* O MultiPay suporta diversas formas de pagamento, incluindo PIX, cartão de crédito em até 6 vezes sem juros, cartão de débito, Mercado Pago e PicPay. Com essa variedade de opções, você pode oferecer aos seus clientes diferentes métodos de pagamento para tornar a experiência de compra mais conveniente.
* [Multifidelidade](https://multicrm.multifidelidade.app/Multi/CriarConta) usa o que se tem de mais moderno em tecnologia antifraude em tempo real, como o protocolo 3DS 2.0, e uma autenticação de cartão via código de segurança na fatura do cliente, tudo para garantir 100% do recebimento dos pagamentos para você empresário. Venda online sem se preocupar com fraudes.

= 5. Como recebo os pagamentos provenientes do MultiPay? =

* Para receber os pagamentos provenientes do MultiPay, você precisará fornecer os seus dados bancários durante o processo de cadastro na Multifidelidade. Certifique-se de inserir corretamente as informações bancárias, para que os pagamentos sejam depositados na sua conta bancária indicada. 

= 6. Qual é o prazo para receber os pagamentos? =

* O prazo para receber os pagamentos depende dos termos e condições da Multifidelidade. Geralmente, os pagamentos são processados em até 14 dias e transferidos para a conta bancária cadastrada. Verifique os detalhes do seu contrato com a Multifidelidade para obter informações mais precisas sobre os prazos de pagamento.

= 7. O MultiPay oferece suporte técnico? =

* Sim, estamos aqui para ajudar! Se você tiver qualquer dúvida ou encontrar problemas técnicos com o MultiPay, entre em contato com nossa equipe de suporte. Você pode enviar um e-mail para suporte@multifidelidade.com.br ou visitar a nossa página
